Auto Shutdown Help Documentation

Getting a License Key

To use Auto Shutdown, you need a valid license key from the EskomSePush API.
You’ll be prompted to enter this key the first time you search for an area. The key is free for personal use and only needs to be obtained once. Follow these steps to get your key:

  1. Visit the EskomSePush API page.
  2. Select the free plan (under the "Monthly" payment option).
  3. Complete the subscription process (you won’t be charged).
  4. Copy your license key (format: XXXXXXXX-XXXXXXXX-XXXXXXXX-XXXXXXXX).

Navigating the Interface

1. Power Button

Toggles the program on or off:

Green: Enabled
Red: Disabled

2. Area Selection

Displays the currently selected area. Click to search for a new one. (Use the same area name format as in the ESP mobile app, e.g. "Sandbaai (6)".)

3. Upcoming Events

Shows the next scheduled loadshedding time. This updates automatically based on the selected area.

4. Shutdown Timing

Choose how long before loadshedding starts your system should shut down. You can select a preset or enter a custom time (in minutes).

5. Next Shutdown

Displays the exact time your system will shut down based on your selected timing and the upcoming loadshedding event.

Error Codes

Error Code Description
403 Not Authenticated (Invalid or disabled token)
404 Not Found (Area or event not available)
408 Request Timeout (Try again later)
429 Too Many Requests (API quota exceeded)
5xx Server-side Error (Please contact ESP support)
