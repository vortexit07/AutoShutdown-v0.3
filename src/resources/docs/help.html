Auto Shutdown Help Documentation

Getting a License Key

  1. Visit the EskomSePush API Page.
  2. Select the free plan.
  3. Complete the subscription (you won't be charged).
  4. Copy your license key (format: XXXXXXXX-XXXXXXXX-XXXXXXXX-XXXXXXXX).

Navigating the Interface

1. Power Button

Toggles the functionality of the program:

Green: Enabled
Red: Disabled

2. Area Selection

Shows the current selected area. Click on it to search for a new area.

3. Upcoming Events

Displays the next loadshedding time and updates automatically based on the selected area.

4. Shutdown Timing

Choose how long before loadshedding starts to shut down your system. Select a preset or enter a custom time in minutes.

5. Next Shutdown

Shows the calculated shutdown time based on your timer settings and the upcoming loadshedding event.

Error Codes

Error Code Description
403 Not Authenticated (Invalid or Disabled Token)
404 Not Found (Area or Event Missing)
408 Request Timeout (Try Again Later)
429 Too Many Requests (API Quota Exceeded)
5xx Server-side Error (Contact Support)
